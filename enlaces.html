Enlaces o links

Se usa la etiqueta a y contiene el atributi obligatorio href.

Otro atributos de esta etiqueta son. target y donload.

Target

define donde se abrirá el recurso solicitado. Por norma general se usa el valor "_blank" para rutas absolutas. Su valor por defecto es "_self".

Donwload

patributo booleano que sirve para descargar el recurso solicitado. Este debe estar en tu mismo servidor
