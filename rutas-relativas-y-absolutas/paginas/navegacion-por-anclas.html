Navegación con anclas

Se realiza la nevegacion dentro de la misma página, yendo a sus diferentes secciones directamente. Se usa mucho en blogs.

Se realiza colocando un id a la etiqueta al cual se dirijirá el enlace. Para hacer referencia a ese id desde el href se inicia con el símbolo #

Iclusive se puede utilizar para nevgar entre las páginas del sitio web al incluir el numerar más el id en la ruta relativa.
