ATRIBUTOS

Son valores adicionales que configuran los elementos y ajustan su comportamiento.

Atributos comunes

Sintaxis:

< etiqueta atributo= "valor" >< / etiqueta >

Ejemplo: < meta charset="utf-8"/>

Atributos booleanos

No llevan valor... Sintaxis:

< etiqueta atributo >< / etiqueta >

Algunos stributos globales disponibles para html

class (clases)

Se usa para dar estilos a través de css. El color azul de este párrafo se dio por medio de la clase p1 en la etiqueta p y css.

id (identificador)

Es un identificador único (no se puede repetir) que se utiliza para seleccionar el elemento desde JS y para hacer navegación a través de anclas

title (título con tooltip)

Es un atributo que ayuda a la accesibilidad mostrando una descripción del elemento al que pertenece. Se visualiza al colocar el cursor (mouse) sobe el elemento.

data (datos pasados en la etiqueta)

Es un atributo que nos permite guardar algún valor en la ertiqueta html. Se trabaja con JS.
