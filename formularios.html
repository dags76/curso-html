Algunos input type

text

Es el valor por defecto... para introducir textos

input type para trabajar con botones

button

Es igual al botón enviar visualmente, pero no envìa formularios

submit

Se utiliza para enviar formularios

reset

resetea el formulario. Se suele usar al lado del submit

type input para trabajar con fechas

date

campo para introducir fechas

time

Para introducir una hora

datetime-local

fechas y horas. Combina input date con input time... pero no funciona en firefox. Es por ello que se recomienda usar los type input date y time para tal fin

mont

Para intrucir un mes

week

Para selecionar una semana

inputs type de uso frecuentes

search

Se utiliza para las barras de busqueda

tel

Npúmero telefónico

pasword

contraseña

email

correo electrónico

url

Dirección web

number

Solo admite valores numéricos

Type inputs varios

range

Muestra una barra para seleccionar un rango. Tiene dos atributos: step y min y max

color

Para seleccionar un color

hidden

Campo oculto, puede contener valores, pero no se mostrarán.

type inputs de tipo seleccionables

radio

Permite seleccionar una única opción de una lista de opciones relacionadas. atributos : name con el mismo valor para todas los inputs, value para indicar el valor de cada input en particular, checked para marcar oor defecto una opción si se amerita.

Indica tu género:

checkbox

Permite seleccionar varias opciones de una lista de opciones relacionadas. Loa atributos name, value (siempre van en todos los inputs de cualquier tipo) y checked.

Indica cuál lenguaje conoces

select

Realmente es una etiqueta a parte, pero permite seleccionar una o varias opciones de una lista.

Cada opción irá dentro de una etiqueta option.

Sin muchas opciones se pueden agrupar mediante la etiqueta opgroup con el atributo label para nombrar la categoría.
