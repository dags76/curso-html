type inputs de tipo seleccionables

radio

Permite seleccionar una única opción de una lista de opciones relacionadas. atributos : name con el mismo valor para todas los inputs, value para indicar el valor de cada input en particular, checked para marcar oor defecto una opción si se amerita.

Indica tu género:

checkbox

Permite seleccionar varias opciones de una lista de opciones relacionadas. Loa atributos name, value (siempre van en todos los inputs de cualquier tipo) y checked.

Indica cuál lenguaje conoces

list

desplega una lista seleccionable. Al input añadir el atributo list. Este type input funciona con la etiqueta datalist, el cual se enlaza al imput mediante el atributo id con el mismo valor. Dentro de esta se anida la etiqueta option la cual mostrará la opción mediante el atributo value

Elige tu mascota favorita:

select

Realmente es una etiqueta a parte, pero permite seleccionar una o varias opciones de una lista.

Cada opción irá dentro de una etiqueta option.

Sin son muchas opciones se pueden agrupar mediante la etiqueta opgroup con el atributo label para nombrar la categoría. se usa en la etiqueta select el atributo multiple, lo cual permite al usuario , usando la tecla ctrl hacer selección multiple.

Type inputs varios

range

Muestra una barra para seleccionar un rango. Tiene dos atributos: step y min y max

color

Para seleccionar un color

hidden

Campo oculto, puede contener valores, pero no se mostrarán.

Otros elementos o etiquetas en los formularios

fielset y legend

fielset se usa para agrupar elementos dentro de un formulario y legend para darle nombre a la misma

Datos personales

meter

representa un valor dentro de un rango conocido. Atributos: min - max - value - low - high - optimus. Los últimos tres marca los límites con colores.

progress

De poco uso. representa el progreso de una tarea.

textarea

Este es muy común. Se usa para introducir texto en un formulario.

Atributos principales o más usados en formularios

placeholder

Da una pista de lo que el usuario tiene que introducir

required

Hace que el campo sea obligatorio. Salta un alerta al clickear el botón de enviar.

readonly

Hace que el campo sea solo de lectura. Se selecciona, pero no se puede escribir ni borrar. El campo se envia.

disable

Desactiva el campo, no se podrá escribir en el ni se enviará el campo.

min-max

Estable el mínimo y el máximo de un campo numérico. Se pueden usar individualmente o ambos a la vez.

minlenght-maxlenght

Establece el mínimo y el máximo de caracteres de un campo de texto. No se envía hasta alcanzar el mínimo, y no deja pasarse del máximo de caracteres establecido.

selected

Equivale a los checked, establece una opción por defecto en las etiquetas select.
