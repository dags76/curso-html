listas anidadas

Se pueden anidar una lista dentro de otra, ya sea una ul dentro de otra o bien sea una ol dentro de otra. También una ol dentro de una ul y viceversa.

ejemplo

Índice

        <ol>
            <li>HTML
                <ol>
                    <li>Estructura</li>
                    <li>Sintaxis</li>
                    <li>Etiquetas</li>
                </ol>
            </li>
            <li>CSS</li>
            <li>JS</li>
        </ol>
  1. HTML
    1. Estructura
    2. Sintaxis
    3. Etiquetas
  2. CSS
  3. JS

Atributos

ol

TYPE: Es el estilo de la numeración: 1, A, a, I, i

        <ol type="A">
            <li>HTML
                <ol type="a">
                    <li>Estructura</li>
                    <li>Sintaxis</li>
                    <li>Etiquetas</li>
                </ol>
            </li>
            <li>CSS</li>
            <li>JS</li>
        </ol>
  1. HTML
    1. Estructura
    2. Sintaxis
    3. Etiquetas
  2. CSS
  3. JS

START: Establecer el inicio de la secuencia. Se coloca el número desde donde va a comenzar.

        <ol type="A" start="4">
            <li>HTML
                <ol type="6" start="f">
                    <li>Estructura</li>
                    <li>Sintaxis</li>
                    <li>Etiquetas</li>
                </ol>
            </li>
            <li>CSS</li>
            <li>JS</li>
        </ol>
  1. HTML
    1. Estructura
    2. Sintaxis
    3. Etiquetas
  2. CSS
  3. JS

ul

TYPE: Establecer el estilo de los items. disc (por defecto), square, circle.

        <ul type="circle">
            <li>HTML
                <ul type="square">
                    <li>Estructura</li>
                    <li>Sintaxis</li>
                    <li>Etiquetas</li>
                </ul>
            </li>
            <li>CSS</li>
            <li>JS</li>
        </ul>
